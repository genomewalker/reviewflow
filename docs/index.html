01 — CAPABILITIES

Built for academic workflows

Every feature designed around how researchers actually work with peer review feedback.

01

AI-assisted drafts

Generate contextual response drafts using Claude or GPT. Multiple strategies per comment—from quick acknowledgments to detailed expansions.

02

Expert panel

Virtual domain experts discuss complex technical questions. Get methodology, statistical, and writing perspectives on difficult comments.

03

Progress tracking

Visual dashboard across all reviewers. See pending, in-progress, and completed at a glance. Never lose track of where you are.

04

Smart import

Parse reviewer comments from PDF, DOCX, or plain text. Automatic categorization by reviewer, type, and priority level.

05

Multi-paper

Handle multiple manuscripts simultaneously. Each paper has its own workspace, context, and progress state.

06

Export to Word

Generate formatted response documents ready for journal submission. Professional layout with one click.

Up and running in minutes

Requires Node.js 18+. Works on macOS, Linux, and Windows.

# Install OpenCode CLI (for AI features) $ npm i -g opencode-ai # Install Rebuttr $ npm i -g github:genomewalker/rebuttr # Launch $ rebuttr
