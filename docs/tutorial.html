Getting Started Tutorial

Learn how to use ReviewFlow to manage your peer review responses from start to finish.

1 Installation 2 Add Paper 3 Process 4 Review 5 Export
1

Installation

Install ReviewFlow and its dependencies. This takes about 2 minutes.

Terminal Commands

  1. First, install the OpenCode CLI (required for AI features):
    npm i -g opencode-ai
  2. Then install ReviewFlow:
    npm install -g reviewflow
  3. Verify the installation:
    reviewflow --help
Tip

On macOS, you may need Xcode Command Line Tools. Run xcode-select --install if you see compilation errors.

2

Add Your Paper

Import your reviewer comments into ReviewFlow. You can paste text, upload files, or use the CLI.

The Add Paper dialog in the web interface

Steps

  1. Launch ReviewFlow: reviewflow
  2. Click "Add Paper" button (top right)
  3. Enter your paper title and metadata
  4. Choose how to import reviews:
    • Paste text: Copy/paste reviewer comments directly
    • Upload file: Drop a PDF, DOCX, or TXT file
    • Import JSON: Use structured JSON format
  5. Click "Process" to parse the reviews
PDF Support

Upload decision letters directly from journals.

Word Documents

Parse DOCX files with reviewer comments.

JSON Format

Use structured JSON for programmatic import.

3

Process with AI

Let AI analyze your comments and generate expert discussions with potential solutions.

AI processing analyzes each comment and generates responses

Steps

  1. Click the "Process Paper" button in the dashboard
  2. Watch the progress as AI analyzes each comment:
    • Categorizes comment type (major/minor)
    • Identifies key concerns
    • Generates expert panel discussion
    • Creates potential solutions
  3. Processing runs in the background - you can continue working
What the AI Generates

For each comment, you'll get: an expert panel discussion, recommended response, advice to the author, and 2-3 alternative response strategies with different effort levels.

4

Review & Edit Responses

Browse AI-generated responses, select from strategies, and refine your answers.

The comment detail view with AI insights and response options

Steps

  1. Click any comment to view its detail page
  2. Review the Expert Panel discussion for insights
  3. Choose from Potential Solutions:
    • Low effort: Quick acknowledgment
    • Medium effort: Detailed explanation
    • High effort: New analysis or revision
  4. Click "Use This" to apply a solution as your draft
  5. Edit the draft response in the text area
  6. Mark as Complete when finalized
Important

Always review and edit AI-generated responses. They are drafts to help you get started, not final answers.

5

Export Your Responses

Generate a formatted document with all your responses, ready for journal submission.

Export options in the dashboard

Steps

  1. Go to the Dashboard view
  2. Click "Export" dropdown
  3. Choose your format:
    • Word Document: Formatted DOCX file
    • Plain Text: Simple TXT file
    • JSON: Structured data for backup
  4. Review the generated document
  5. Submit with your revised manuscript!
Pro Tip

Export to JSON periodically to backup your progress. You can reimport this file to restore your work.
